# Техническое задание

## Название проекта

**Social Media Analytics Dashboard**

---

# Цель проекта

Разработать современное веб-приложение (Dashboard), которое автоматически собирает статистику видеороликов из Instagram и TikTok по ссылкам и отображает общую аналитику в режиме реального времени.

Приложение предназначено только для владельца проекта.

Добавление новых видео выполняется исключительно через Claude Code (или напрямую через сервер/API). На самом сайте обычные пользователи ничего добавлять, удалять или редактировать не могут.

---

# Основной функционал

## Добавление видео

Система должна принимать ссылки на:

* TikTok
* Instagram Reels

Например:

https://www.tiktok.com/...

https://www.instagram.com/reel/...

После добавления ссылки приложение автоматически:

* определяет платформу;
* получает всю доступную информацию;
* сохраняет данные в базе.

---

# Какие данные необходимо получать

Для каждого видео хранить:

### Основная информация

* ссылка
* платформа (TikTok / Instagram)
* название аккаунта
* ник аккаунта
* аватар аккаунта
* дата публикации
* описание видео
* обложка видео
* ссылка на само видео

---

### Статистика

Для каждого видео необходимо отображать:

* количество просмотров
* количество лайков
* количество комментариев
* количество репостов (если доступно)
* количество сохранений (если возможно получить)
* дата последнего обновления статистики

---

# Автоматическое обновление

Каждый раз при открытии сайта система должна автоматически:

* обновить статистику всех видео;
* получить актуальные лайки;
* актуальные просмотры;
* актуальные комментарии.

Также предусмотреть фоновое обновление каждые 10–15 минут.
